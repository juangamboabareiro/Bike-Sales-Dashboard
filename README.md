# Bike Sales Dashboard


## Descripción

Este proyecto consiste en un análisis de ventas de bicicletas. El objetivo es transformar datos crudos mediante limpieza de datos, tablas dinámicas y visualizaciones.



## Objetivos

- Analizar el comportamiento de ventas de bicicletas
- Identificar patrones por región, tipo de cliente y producto


## Dataset

Fuente: dataset de ventas de bicicletas

Tipo de datos:
- Información demográfica de clientes
- Datos de compra
- Características del producto



## Proceso de análisis

### Limpieza de datos
- Normalización de categorías
- Corrección de valores inconsistentes
- Eliminación de duplicados

### Transformaciones
- Creación de variables categóricas
- Segmentación de clientes

### Análisis exploratorio
- Uso de tablas dinámicas
- Comparación entre segmentos

### Visualización
- Gráficos dinámicos
- Slicers





## KPIs y métricas
- Ventas totales
- Cantidad de clientes
- Ventas por región
- Ventas por categoría de producto
- Segmentación por características del cliente



## Herramientas

### Microsoft Excel
- Tablas dinámicas
- Gráficos dinámicos
